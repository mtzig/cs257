About

    This website was created by Thomas Zeng and Cole Weinstein for Carleton's CS257 final project. It was built using references from w3schools, Mozilla Developer Network Web Docs, and Jeff Ondich's sample html pages.

    The original data was from UNESCO. However, we got the data from Kaggle which scraped from The Guardian which scraped from UNESCO.
